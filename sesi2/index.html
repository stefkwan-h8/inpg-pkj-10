Atribut Tag HTML

tag html bisa polos, sendirian, bisa juga punya atribut. contohnya tag img punya atribut src dan alt. tag input punya atribut type. form punya action. a punya href.

Kita bisa menambahkan atribut ke sebuah tag html. ada atribut yang dimiliki dan bisa ditambah ke semua tag html yaitu: id dan class.

id ibarat KTP, id sebuah tag html harus unik, tidak boleh sharing nama id yang sama dengan elemen html lain.

class ibarta grup, wa grup, kelas, batch, class yang sama bisa di miliki banyak elemen html berbeda. satu elemen html bisa punya banyak class.

Sedikit background tentang CSS

menambahkan atribut style kedalam sebuah elemen HTML membuat kita bisa menulis CSS di dalamnya dan memberi style ke elemen tersebut. Tetapi, ini adalah cara menulis CSS yang paling BURUK. Selain sulit untuk diatur saat sudah banyak HTML tag dan CSS nya, sudah di update, susah di track, juga sangat berantakan. Biasanya para professional web developer akan membuat file CSS berbeda khusus untuk tempat tulis CSS. HTML nya bersih dari CSS

Cara menulis CSS yang sedikit lebih baik adalah dengan tag style di dalam head.

Cara terbaik, paling rapi, paling professional adalah dengan membuat file CSS terpisah, yang kemudian di link dengan file HTML kita.

Cara pemakaian CSS

cara nulis css di html ada tiga cara: langsung di tag html pakai atribut style, pakai tag style di head, pakai file .css terpisah yang kemudian di link dengan tag link di head

CSS sendiri bisa styling 3 hal berbeda: tag html, id, class

property CSS yang bisa di styling ada banyak, termasuk: font, warna font, warna background, size font, ukuran, border, margin, padding, ...

ada elemen jenis inline yang bisa sharing satu baris, seperti img. img, kalau muat, bakal nempel2 sama img berikutnya di baris yang sama.

teman div yang sama2 tidak punya arti tertentu adalah span. bedanya, kalau div elemen Blok, span adalah elemen inline. misal ada yang mau aku pisahkan di sebuah blok text di dalam paragraph, bisa pakai span. kalau pakai div dia akan pisahin dan kasih banyak space, karena dia ambil seluruh baris. kalau span, dia ambil secukupnya saja.
