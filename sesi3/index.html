CSS Tag Selector

Sebuah penanda, elemen/tag HTML yang ingin diberi style. Anda dapat menuliskan langsung nama tag yang ingin diberi style tanpa perlu menambahkan tanda

CSS cara penulisan

cara penulisan CSS bisa dengan:

1. langsung di tag HTML pakai atribut style

2. di dalam tag style di HTML

3. cara file terpisah .css yang disambung ke HTML dengan tag link rel="stylesheet" href="file.css" di dalam head nya HTML

property CSS

halaman dokumentasi CSS di w3schools

  • color untuk warna text
  • background-color untuk warna background
  • font-family untuk ganti pilihan font
  • font-style text italic atau tidak
  • font-weight text seberapa bold
  • margin spasi antara elemen dengan elemen lainnya
  • padding spasi antara konten elemen dengan border elemen
  • border memberikan garis di sekeliling elemen HTML
